Orientação a Objetos

No JS, tudo é objeto! 

<script type='text/javascript'>

	// dados primitivos

	var numero = 1;
	
	numero 			// 1
	numero.valueOf(); 	// 1
	numero.toString();	// "1"
	
	
	// funções
	
	var funcao = function () { alert("funcao foi chamada"); };
	
	funcao(); 		// alert: funcao foi chamada
	funcao.toString(); 	// "function () { alert("funcao foi chamada"); }"

</script'>

Orientação a Objetos

  • Objetos com membros públicos e privados

Namespaces

Mais esse no javascript?!

Namespaces


O que é?

  • Técnica empregada para evitar colisões de nomes de classes em um contexto global.
  • É a mesma idéia do namespace no C# e do pacote no Java.

Por que usar?

  • Separa as classes em grupos de mesmo interesse.
  • Facilita a manutenção.

Namespaces


Como usar?

  • Através de Objetos Literais
  • Estruturação de pastas e arquivos

Objetos Literais


O que são?

  • Grosseiramente falando, um conjunto de pares nome/valor separados por vírgula que estão contidos num bloco.
  • Nome: qualquer nome válido no JS
  • Valores podem ser: dados primitivos, métodos e outros objetos literais.

Objetos Literais

Exemplo

<script type='text/javascript'>

	// objeto literal

	var pessoa = {
		nome: 'Fulano',
		idade: 25,
		endereco: {
			rua: 'Av. São Paulo',
			numero: 1000,
			bairro: 'Centro',
			toString: function () {
				alert(this.rua + ', ' + this.numero + ' - ' + 
				      this.bairro);
			}
		},
		toString: function () {	alert(this.nome + ', ' + this.idade); }
	};

</script'>

Objetos Literais

Exemplo

<script type='text/javascript'>

	// acessando propriedades

	pessoa.nome		// "Fulano"
	pessoa.idade		// 25	
	pessoa.endereco.rua	// "Av. São Paulo"
	
	
	// acessando metodos
	
	pessoa.toString();		// alert: Fulano, 25
	pessoa.endereco.toString();	// alert: Av. São Paulo, 1000 - Centro

</script'>

Implementando um namespace

// Objeto global - Namespace raiz

MyApp = {
	Component: {},	// Namespace para plugins

};

Namespaces

Objeto que encapsula um plugin qualquer

// Objeto que encapsula um plugin qualquer

MyApp.Component.Tabela = function (idElemento) {	
	this.tabela = $('#' + idElemento).dataTables();	
};

MyApp.Component.Tabela.prototype.limparTabela = function () {		
	this.tabela.fnClearTable();
};

Namespaces

Configuração de comportamentos

// Objeto que configura os comportamentos de uma página

MyApp.Script.UsuarioScript = function () {

	var that = this;
	var tabela = new MyApp.Component.Tabela("usuario-grid");

	that.limparTabela = function (event) {
		event.preventDefault();
		tabela.limparTabela();
	};
};

$(function () {	
	var script = new MyApp.Script.UsuarioScript();
	
	// Ligações de eventos
	
	$('a.limparTabela').click(script.limparTabela);	
});
